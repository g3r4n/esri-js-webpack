# esri-js-webpack

## Features

* Arcgis JS API webpack plugin
* React 16
* Webpack 4
* Babel
* Prettier
* Staging ES Next Features
* Hot Module Replacement

## Installation

```
git clone git@github.com:g3r4n/esri-js-webpack.git
cd esri-js-webpack
npm install
npm run start
```

visit `http://localhost:8080/`

# build

```
npm run build
```

# serve build

```
npm run serve
```

# lint code using prettier

```
npm run lint
```
